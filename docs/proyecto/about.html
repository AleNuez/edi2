Idea Principal

Nombre del Proyecto:

ConcursAR

Descripción:

Concursar es una plataforma virtual de Recursos Humanos del sistema educativo nacional.

Objetivo:

Tiene como proposito unir horas o cargos disponibles de los postulantes con horas vacantes o cargos disponibles en diferentes instituciones educativas.

Justificación:

Se plantea este proyecto como innovación al sistema actual de actos públicos y concursos docentes donde los procedimientos son prescenciales y gran parte de la gestión no es digital.

Tiempo:

El proyecto abarcará 2 meses de producción. siendo este finalizado para el 22/06.

Caracteristicas

Metodología (Sistema General):

La metodología del sistema sería una migración digital del sistema actual de concursos docentes, con el sistema de puntajes actualmente en vigencia.

Usuario final (¿Para quién está destinada la plataforma?):

Principalmente para postulantes del espectro docente, esto incluye: Maestros, Profesores, Preceptores. Luego Administradores del sitio, Esto es: Secretarios de escuelas y personal de la Secretaria de Asuntos Docentes.

Alcance (¿Qué nivel institucional y cuán escalable es?):

Distrital, para la educación Publica de Jardines, Escuelas Primarias, Secundarias, Escuela Técnica, Escuela Especial y Modalidad Adultos. El sistema es Nacional, dividiendose por Provincias y Distritos.

El sistema integra la Educación Privada. ofreciendo cargos disponibles como recomendación (esto funciona como publicidad).

Tecnologías a utilizar:

  • Hosting: 000Webhost.
  • FTP: Filezilla.
  • Frontend: HTML, CSS (Flexbox) , Javascript (sin Frameworks).
  • Backend: PHP
  • Base de datos: SQLserver.
  • Proyecto: Trello.
  • Gestión de versiones: GIT, Github.
